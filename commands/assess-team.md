# /assess-team - Team Assessment & Health Check

Evaluate your team's health, dynamics, performance, engagement, and get actionable recommendations for improvement.

## Usage

```
/assess-team
/assess-team [dimension]
/assess-team detailed
/assess-team compare
```

## Assessment Dimensions

### Team Health (Overall)
```
/assess-team health
```
Evaluates:
- Psychological safety
- Engagement and morale
- Communication quality
- Collaboration effectiveness
- Conflict management
- Growth mindset

### Performance & Productivity
```
/assess-team performance
```
Measures:
- Goal achievement
- Quality of work
- Velocity and throughput
- Delivery on time
- Technical excellence
- Metrics and KPIs

### People & Culture
```
/assess-team culture
```
Assesses:
- Team culture alignment
- Inclusion and belonging
- Recognition and appreciation
- Work-life balance
- Retention and attrition
- Team rituals and traditions

### Team Dynamics
```
/assess-team dynamics
```
Evaluates:
- Trust and safety
- Collaboration patterns
- Conflict resolution
- Leadership effectiveness
- Diversity of perspectives
- Team cohesion

### Skills & Growth
```
/assess-team skills
```
Measures:
- Technical skill level
- Learning capability
- Development progress
- Skill gaps
- Growth opportunities
- Leadership pipeline

### Engagement & Satisfaction
```
/assess-team engagement
```
Assesses:
- Overall engagement
- Job satisfaction
- Sense of purpose
- Career development
- Manager relationship
- Company alignment

## How Assessment Works

### 1. Comprehensive Evaluation
- Structured questions
- Multiple dimensions
- Quantitative and qualitative
- Comparison to benchmarks
- Trend analysis

### 2. Detailed Findings
- Strengths (what's working)
- Growth areas (what needs improvement)
- Risks and opportunities
- Comparison to previous assessments
- Industry benchmarks

### 3. Actionable Recommendations
- Specific improvements
- Implementation plan
- Priority sequencing
- Resource needs
- Success metrics
- Timeline

### 4. Detailed Report
- Executive summary
- Dimension-by-dimension analysis
- Radar chart visualization
- Comparison trends
- Recommended actions
- Owner and timeline

## Assessment Types

### Quick Assessment (15 min)
```
/assess-team quick
```
Fast snapshot of team health across key areas

### Detailed Assessment (30 min)
```
/assess-team detailed
```
Comprehensive evaluation with specific recommendations

### Focused Assessment (20 min)
```
/assess-team [dimension]
```
Deep dive on specific area (performance, culture, etc.)

### Comparative Assessment
```
/assess-team compare
```
Compare current state to:
- Previous assessments
- Team goals
- Industry benchmarks
- Other teams

## Recommended Assessment Cadence

- **Ongoing**: Regular 1-on-1s and observations
- **Monthly**: Quick pulse check
- **Quarterly**: Detailed assessment
- **Annually**: Comprehensive evaluation + planning

## Using Assessment Results

### 1. Share with Team
```
- Summarize key findings
- Get feedback on accuracy
- Discuss recommendations
- Collaborative improvement planning
```

### 2. Create Action Plan
```
- Prioritize improvements
- Assign owners
- Set timelines
- Define success metrics
```

### 3. Track Progress
```
- Monthly check-ins
- Quarterly assessment
- Celebrate progress
- Adjust plan as needed
```

### 4. Continuous Improvement
```
- Learn from results
- Adapt based on feedback
- Share learnings
- Build culture of continuous improvement
```
