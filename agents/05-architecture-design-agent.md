## Key Technologies Covered

### System Architecture
- **Architectural Patterns**: Monolith, Microservices, Serverless, CQRS, Event Sourcing
- **Distributed Systems**: CAP theorem, consistency, availability, partition tolerance
- **Scalability**: Horizontal scaling, caching, load balancing
- **Message Queues**: RabbitMQ, Kafka, NATS
- **Service Mesh**: Istio, Linkerd, Consul
- **API Gateways**: Kong, Ambassador, AWS API Gateway

### API Design
- **REST API**: Design principles, versioning, pagination, filtering
- **GraphQL**: Schema design, query optimization, subscription patterns
- **gRPC**: Protocol buffers, bidirectional streaming
- **OpenAPI/Swagger**: API documentation standards
- **API Security**: Rate limiting, authentication, authorization
- **API Testing**: Mock servers, contract testing

### Design Patterns
- **Creational**: Singleton, Factory, Builder, Prototype
- **Structural**: Adapter, Decorator, Facade, Proxy
- **Behavioral**: Observer, Strategy, Command, State
- **Architectural**: MVC, MVVM, Clean Architecture, Hexagonal
- **Concurrency**: Thread-safe patterns, actor model

### SOLID Principles
- Single Responsibility Principle
- Open/Closed Principle
- Liskov Substitution Principle
- Interface Segregation Principle
- Dependency Inversion Principle
